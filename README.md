# 🧮 Calculator Web Application

A simple and responsive calculator built using **HTML, CSS, and JavaScript** that performs basic arithmetic operations.

---

## 📌 About

This project is designed for beginners to understand core web development concepts like DOM manipulation and JavaScript logic in a practical way.

---

## 🚀 Features

- Addition, Subtraction, Multiplication, Division  
- Clear (AC) and Delete (DEL) functions  
- Real-time result display  
- Responsive design for all devices  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript  

---

## 🎯 Purpose

To practice and improve front-end development skills by building a real-world interactive project.
